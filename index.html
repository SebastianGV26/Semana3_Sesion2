Que es CSS y para que Sirve

Es un lenguaje de hoja de estilos que permite crear el diseño de una web, este lenguaje sirve para personalizar y darle mejor vista a la página web y así poder presentar de una manera agradable el contenido al usuario

Maneras de enlazar CSS

se puede hacer mediante el link en el head, inline, con el atributo style de HTML o vinculando con el import

Ejemplos

En el head: "<"link rel="stylesheet" href="./styles.css">"

Inline: "<"h1 style="color: red;"> Hola Mundo"<"/h1">"

"<"style type = "text/css">
p{color: red;}
"<"/style">"

@import "./styles.css"

Los pilares de CSS

Selectores

Son la manera de indicar a que elementos de html se le va aplicar estilos

Ejemplos

Como por ejemplo: h1, h2, p, a

Cascada

Significa que el último valor es el que prevalece, siempre y cuando tenga la misma especificidad

Ejemplos

Como por ejemplo: h1{color: red;}

h1{color: blue}

El texto aparecerá de color azul, ya que es el último valor que se le dio al selector

Herencia

Es cuando el selector hereda valores del padre con el valor inherit

Ejemplos

h3{color: skyblue;}

p{color: inherit;}

El parrafo aparecerá de color celeeste, ya que su padre tiene ese valor

Especificidad

Es número o valor que adquieren los selectores para que puedan ganar jerarquía y así poder tener los estilos css del mayor, no hace caso a la cascada.

"<"h3 class="especificidad" id="especi" ">"Hola Mundo "<"/h3>

h3.especificidad{ color: skyblue; }

h3#especi{ color: blue; }

El h3 tomará el color azul, ya que por especificidad, el id tiene el valor más alto y por ende, prevalece






* El important puede ser superado por otro important con mayor valor de espeficidad, sin importar la cascada
